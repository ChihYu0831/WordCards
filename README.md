# WordCards

## 專案簡介

本專案是一個英文單字學習系統，主要提供單字瀏覽、發音播放、單字編輯與單字測驗功能。使用者可以透過左側清單選擇不同單字，查看單字本身、音標與中文解釋，並播放對應的音檔來練習發音。

系統也提供自動播放功能，可以依序播放單字，方便使用者進行聽力與單字複習。此外，使用者可以雙擊單字進入編輯畫面，修改單字、音標、音檔路徑與解釋內容，修改後會儲存回單字資料檔。

本專案另外加入單字測驗模式，系統會隨機產生題目，讓使用者根據音標選出正確的英文單字，並即時計算答題分數，提升單字記憶與辨識能力。

---

## 功能特色

1. 顯示英文單字、音標與中文解釋。
2. 點選單字後可播放對應的英文發音音檔。
3. 支援自動播放單字功能。
4. 可使用 Enter 鍵切換下一個單字。
5. 可使用 Space 鍵重複播放目前單字。
6. 支援單字資料編輯功能。
7. 編輯後可將資料儲存回單字檔案。
8. 提供單字測驗模式，根據音標選出正確英文單字。
9. 測驗模式會即時計算答對題數與總作答題數。

---

## 執行說明

### 單字瀏覽
1. 啟動程式後，左側會顯示單字清單。
2. 點選任一單字後，右側會顯示該單字的英文、音標與中文解釋。
3. 若該單字有對應音檔，程式會播放該單字發音。

### 自動播放
1. 按下 Play 按鈕後，程式會開始自動播放單字。
2. 播放時會依照單字清單順序切換到下一個單字。
3. 再次按下按鈕後，會停止自動播放。

### 鍵盤操作
1. Enter	播放下一個單字
2. Space	重複播放目前單字

### 編輯單字
1. 在左側單字清單中雙擊想要修改的單字。
2. 系統會開啟「編輯單字」視窗。
3. 可修改以下內容：
   - 單字
   - 音標
   - 音檔路徑
   - 中文解釋
4. 按下「儲存」後，資料會更新並儲存回單字檔。

### 測驗模式
1. 按下主畫面中的 `Test` 按鈕。
2. 系統會開啟單字測驗視窗。
3. 畫面上方會顯示音標。
4. 使用者需要從四個選項中選出正確的英文單字。
5. 答題後系統會顯示答對或答錯，並更新分數。
6. 若單字數量少於 4 個，系統會提醒無法進行測驗。

### 單字資料格式

單字資料儲存在 WordCards.txt 中，每一筆資料包含單字、音標、音檔路徑與中文解釋。程式啟動時會讀取此檔案，並將資料顯示在單字清單中。

範例資料內容：

abacus
`abəkəs
Sound\abacus.mp3
<aba-=abax: 一種計算工具> + <-us: calculus 小圓石>
一種利用小圓石幫助計算的工具
